Frequently Asked Questions

Quick answers to common questions about Codeman.

Codeman is a powerful coding companion app for iPad that helps developers write, edit, and manage code on the go. With its intuitive interface and smart features, it's like having a superhero sidekick for your coding workflow.
Codeman is available for iPad running iPadOS 17.0 or later. We support all iPad models including iPad Pro, iPad Air, iPad mini, and standard iPad.
Navigate to Settings → Connections and tap the "+" button. Enter your server details including hostname, port, and authentication method. Codeman supports SSH key authentication and password-based connections.
Absolutely. All connections are encrypted using industry-standard TLS. SSH keys are stored securely in your device's keychain. We never access, collect, or transmit your code or personal data to our servers.
Codeman uses WebKit for rendering. Most VS Code features and popular extensions work perfectly. Some extensions requiring advanced browser-specific features (like SharedArrayBuffer) may have limited functionality. Core editing, IntelliSense, terminal, Git, and most popular extensions work without issues.
Use the contact form below and select "Bug Report" as the topic. Please include your iPad model, iPadOS version, and steps to reproduce the issue. Screenshots are incredibly helpful!
We love hearing from our users! Send us a message via the contact form with "Feature Request" selected. We review every suggestion and prioritize based on community interest.

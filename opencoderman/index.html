// FEATURES

Everything You Need

A full-featured mobile client for OpenCode—built for developers who work on the go.

Real-Time AI Chat

Send prompts and watch AI responses stream in real-time via Server-Sent Events. See reasoning, tool calls, and code changes as they happen.

Code Review Panel

Review file diffs and changes made by the AI agent with syntax-highlighted diff views. Track additions, modifications, and deletions.

File Explorer

Browse your project’s file tree, open files for viewing, and search across your entire codebase with fuzzy file search.

Multi-Project Management

Organize your work by project with collapsible session groups. Switch between projects effortlessly.

Multiple AI Providers

Connect to OpenAI, Anthropic, GitHub Copilot, Google, OpenRouter, and more. Switch models on the fly with the toolbar model picker.

Build & Plan Modes

Choose between Build mode for code generation and Plan mode for architecture planning. Adjust effort levels from None to Xhigh.

Slash Commands

Use server-defined slash commands like /init, /review, and /create-pr directly from the chat input.

iCloud Sync

Sync server connections, model preferences, and project directories across all your devices via iCloud Key-Value Store.

// FAQ

Frequently Asked Questions

Quick answers to common questions about OpenCoderMan.

OpenCoderMan is a mobile companion app for the OpenCode AI coding platform. It lets you connect to your OpenCode server from your iPad and manage AI-powered coding sessions on the go.
Yes. OpenCoderMan is a client app that connects to a running OpenCode server. You’ll need to install and start an OpenCode server on your development machine, then connect to it from the app via your local network.
OpenCoderMan runs on iPad with iOS 18.0 or later. It supports all orientations on iPad.
OpenCoderMan supports any provider configured in your OpenCode server—including OpenAI, Anthropic, GitHub Copilot, Google, and OpenRouter. You can connect providers via OAuth device code flow directly from the app and switch models on the fly.
OpenCoderMan connects directly to your self-hosted OpenCode server over your local network. No account is required, and no data passes through our servers. Session state is cached locally on your device.
Start your OpenCode server with opencode server, then open the app and go to Settings → Server. Enter the server URL (e.g., http://192.168.1.100:4096) and the app will connect and load your sessions.
Use the contact form below and select “Bug Report” as the topic. Please include your device model, iOS version, and steps to reproduce the issue. Screenshots are incredibly helpful!

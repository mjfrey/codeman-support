Everything You Need

A universal VS Code client optimized for every Apple device—one app, all platforms.

Universal App

One app for iPhone, iPad, and Mac. Buy once, use everywhere. Optimized UI for each platform with adaptive layouts.

VS Code Integration

Works out of the box with vscode.dev—no setup needed. Connect to any custom VS Code server with full desktop mode and chromeless browsing.

Desktop-Class Keyboard

All VS Code keyboard shortcuts work perfectly. Command key passthrough, system shortcuts, and full external keyboard support on iPad.

iPhone Optimized

Touch-friendly interface with quick-access toolbar. Portrait and landscape support. Works great with external keyboards for mobile coding.

iPad Multi-Window

Work on multiple projects with multi-window support. Split View and Slide Over compatible. Perfect for iPad + Magic Keyboard.

Native Mac

Native Mac window management with resizable windows and sensible defaults. Full menu bar integration and all keyboard shortcuts work as expected.

Developer Tools

Built-in debug console with real-time logging. Detailed error messages, connection monitoring, and exportable logs for troubleshooting.

Security & Privacy

HTTPS connections with self-signed certificate support. No tracking, no analytics, no data collection. All settings stored locally on your device.

Frequently Asked Questions

Quick answers to common questions about CodeMan.

CodeMan is a VS Code browser that brings your remote development environment to all your Apple devices. It works out of the box with vscode.dev and connects to any custom VS Code server, code-server instance, or GitHub Codespace.
CodeMan is a universal app for iPhone (iOS 17.0+), iPad (iPadOS 17.0+), and Mac (macOS 14.0+). Buy once and use on all your Apple devices. Each platform has an optimized UI.
No! CodeMan works immediately with vscode.dev—no setup required. If you want to connect to your own server, you can use code-server, VS Code’s built-in server (code --serve-web), or any compatible VS Code server over HTTP/HTTPS.
Yes! All VS Code keyboard shortcuts work perfectly. Command key input is passed through directly to VS Code. System-level shortcuts like Cmd+, (Settings) and Cmd+R (Reload) are handled by the app. External keyboards are fully supported on iPad.
Absolutely. Connections are made directly from your device to your server via HTTPS. We don’t collect, store, or transmit any personal data. There are no analytics or tracking services. All settings are stored locally on your device.
CodeMan uses WebKit for rendering. Most VS Code features and popular extensions work perfectly. Some extensions requiring advanced browser-specific features (like SharedArrayBuffer) may have limited functionality. Core editing, IntelliSense, terminal, Git, and most popular extensions work without issues.
Yes. Enable “Allow Self-Signed Certificates” in Settings to connect to local servers using self-signed certificates.
